# Archevi vs Obsidian

## Overview

Obsidian is a local-first markdown knowledge base. Archevi focuses on family document management with AI search.

## Feature Comparison

| Feature | Archevi | Obsidian |
|---------|---------|----------|
| AI Search | Built-in RAG | Requires plugins |
| Self-Hosted | Yes | Yes (local files) |
| Monthly Cost | $2-15 | Free-$10 |
| Family Sharing | Built-in | Sync addon ($10/mo) |
| Learning Curve | Low | High |
| Note Linking | No | Yes (bidirectional) |
| Graph View | No | Yes |
| Natural Language Query | Primary feature | Via plugins |

## When to Choose Archevi

- You want AI that understands questions
- Easy family sharing is important
- Lower learning curve preferred
- Document search is primary use case
- You want a chat interface

## When to Choose Obsidian

- You love markdown
- You want bidirectional linking
- You need offline access
- You want extensive customization
- You prefer local-first storage
