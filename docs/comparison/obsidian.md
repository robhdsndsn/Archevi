# Archevi vs Obsidian

## Overview

Obsidian is a local-first markdown knowledge base for power users. Archevi is a managed service focused on family document management with AI search built-in.

## Feature Comparison

| Feature | Archevi | Obsidian |
|---------|---------|----------|
| AI Search | Built-in RAG (Cohere) | Requires plugins + setup |
| Hosting | Managed cloud (Canadian) | Local files |
| Monthly Cost | $14.99 CAD | Free-$10 + plugin costs |
| Family Sharing | Built-in | Sync addon ($10/mo/person) |
| Learning Curve | Low (60 seconds) | High |
| Note Linking | No | Yes (bidirectional) |
| Graph View | No | Yes |
| Natural Language Query | Primary feature | Via plugins |
| Voice Notes | Yes (Groq Whisper) | Via plugins |
| OCR | Yes (built-in) | Via plugins |
| Expiry Alerts | Yes | No |

## When to Choose Archevi

- You want AI that understands questions out of the box
- Easy family sharing is important
- Lower learning curve preferred
- Document search is primary use case
- You want a chat interface
- You don't want to manage infrastructure
- Canadian hosting matters (PIPEDA)

## When to Choose Obsidian

- You love markdown and local files
- You want bidirectional linking and graph views
- You need offline access
- You want extensive customization via plugins
- You prefer owning your data files
- You're comfortable with technical setup

## Cost Comparison (Annual, Family of 5)

| Plan | Archevi | Obsidian |
|------|---------|----------|
| Basic | $180 CAD/year | $0 + plugins |
| With Sync | Included | $600/year ($10/mo x 5) |
| With AI | Included | Variable (plugin + API keys) |

## Bottom Line

Archevi is for families who want things to "just work" with AI search included. Obsidian is for technical users who want complete control and are willing to set up and maintain their own system.
