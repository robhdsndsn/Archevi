# Contributing

Help make Archevi better for families everywhere.

## Ways to Contribute

- Report bugs and issues
- Suggest new features
- Improve documentation
- Submit code changes
- Help other users

## Getting Started

1. Fork the repository
2. Set up [development environment](/contributing/development)
3. Make your changes
4. Submit a pull request

## Guidelines

- Follow our [code standards](/contributing/code-standards)
- Write clear commit messages
- Add tests for new features
- Update documentation
