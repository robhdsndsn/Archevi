## Architecture Decision

**Managed Service Only with BYOK (Bring Your Own Key)**

- Platform Fee: Customer pays Archevi for infrastructure ($14.99-$99+/month)
- AI Costs: Customer pays Cohere directly via their own API key (~$2-10/month)

See [00_executive_summary.md](00_executive_summary.md) for full architecture details.

## Key Numbers

- **Target Launch**: 6-8 weeks from start
- **Year 1 Goal**: $50,000 CAD revenue
- **Initial Market**: Canada (Ontario, BC, then Quebec)
- **Customer Target**: 100 paying families by Month 12

## Phase Overview
